# 绘制爱心

用许多随机颜色小窗口拼出一颗爱心图案的趣味程序。
文字可以自定义！

## 功能

- **阶段 1**：用彩色小窗口从左到右、从上到下依次出现，像画画一样一笔一笔拼出爱心轮廓
- **阶段 2a**：爱心绘制完成后，稍作停顿欣赏
- **阶段 2b**：像橡皮擦一样将爱心逐渐擦除
- **阶段 2c**：爱心擦除后，在全屏随机位置持续生成随机小窗口
- **右上角按钮**：一键瞬间关闭全部窗口，退出程序


## 配置参数

在文件顶部 `全局配置` 区域可以调整：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MINI_W, MINI_H` | 36, 22 | 每个小窗口的宽高（像素） |
| `HEART_SCALE` | 1.0 | 爱心缩放倍数 |
| `SPAWN_INTERVAL` | 0.01 | 绘制阶段每个窗口出现间隔（秒） |
| `ERASE_INTERVAL` | 0.008 | 擦除阶段每个窗口消失间隔（秒） |
| `RANDOM_INTERVAL` | 0.12 | 随机窗口生成间隔（秒） |
| `MAX_RANDOM_WIN` | 40 | 全屏随机窗口最多同时存在数量 |

## 算法说明

爱心轮廓采用数学参数方程采样：

```
x = 16sin³(t)
y = -(13cos(t) - 5cos(2t) - 2cos(3t) - cos(4t))
```

对参数 `t` 密集采样 3000 个点，以窗口大小为步长去重，再按 `(y, x)` 升序排序，使窗口从上到下、从左到右依次出现。
